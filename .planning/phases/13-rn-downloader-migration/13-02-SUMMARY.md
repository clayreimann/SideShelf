## Decisions Made

- **TDD RED phase could not commit via pre-commit hook:** Pre-commit hook runs `jest --bail --findRelatedTests`. RED commits (intentionally failing tests) are blocked by the hook. Resolution: proceeded directly to GREEN (Task 3) before committing Task 1, then committed all tasks in order once tests were passing.
- **`^4.5.3` semver range (not exact):** npm adds `^` by default; this satisfies the plan's "no fork URL" requirement. The exact version constraint is enforced by `package-lock.json`.
- **Handlers before `.start()` is a critical invariant:** Documented with a `// CRITICAL:` comment in both `DownloadService.ts` and `rnbd-fork-diff.md`. The `begin` event fires synchronously on `.start()` in some native environments — handlers must be registered first.
