Rewrite `more/index.tsx` to:

1. **Fix navigation bug**: Change `router.push(\`/${tab.name}\`)` → `router.push(\`/more/${tab.name}\`)` for hidden tab items (Series, Authors).

2. **Add `icon` and `isNavItem` fields to `ActionItem`** type:

```typescript
type ActionItem = {
  label: string;
  onPress?: () => void;
  badge?: number;
  styles?: { color?: string };
  icon?: { sf: SFSymbol; ionicon: IoniconsName };
  isNavItem?: boolean;
};
```

3. **Import icons**: Add `SymbolView` from `expo-symbols`, `Ionicons` from `@expo/vector-icons`, `Platform` from `react-native`, `SFSymbol` from `sf-symbols-typescript`, and `ComponentProps` from `react`.

4. **Define `IoniconsName` type** (same as in `(tabs)/_layout.tsx`):

```typescript
type IoniconsName = ComponentProps<typeof Ionicons>["name"];
```

5. **Assign icons per item** in the `data` useMemo. Use these SF Symbol / Ionicons pairs:

| Item                 | SF Symbol                            | Ionicons                |
| -------------------- | ------------------------------------ | ----------------------- |
| Series (hidden tab)  | `square.stack`                       | `layers-outline`        |
| Authors (hidden tab) | `person.circle`                      | `people-circle-outline` |
| About Me             | `person.crop.circle`                 | `person-circle-outline` |
| Settings             | `gearshape`                          | `settings-outline`      |
| Leave Feedback       | `envelope`                           | `mail-outline`          |
| Log out              | `rectangle.portrait.and.arrow.right` | `log-out-outline`       |
| Library Stats        | `chart.bar`                          | `bar-chart-outline`     |
| Storage              | `externaldrive`                      | `server-outline`        |
| Track Player         | `waveform`                           | `radio-outline`         |
| Logs                 | `doc.text`                           | `document-text-outline` |
| Logger Settings      | `slider.horizontal.3`                | `options-outline`       |
| Actions              | `bolt`                               | `flash-outline`         |

6. **Mark nav items** (`isNavItem: true`): Series, Authors (hidden tabs), Settings. About Me, Leave Feedback, Log out are action rows — no chevron.

7. **Render the icon in each row**. In the `renderItem` function, add the icon on the left. Use the same platform pattern as `TabBarIcon` in `(tabs)/_layout.tsx`:

```tsx
{
  item.icon && (
    <View style={{ marginRight: 12 }}>
      {Platform.OS === "ios" ? (
        <SymbolView
          name={item.icon.sf}
          size={20}
          tintColor={colors.textSecondary ?? textSecondary}
          fallback={<Ionicons name={item.icon.ionicon} size={20} color={textSecondary} />}
        />
      ) : (
        <Ionicons name={item.icon.ionicon} size={20} color={textSecondary} />
      )}
    </View>
  );
}
```

8. **Render chevron on nav rows** (right side, before badge):

```tsx
{
  item.isNavItem && (
    <Ionicons name="chevron-forward" size={16} color={textSecondary} style={{ marginLeft: 4 }} />
  );
}
```

9. **Add visible tap state**: Add `android_ripple` and `style` with `pressed` state to the `Pressable`:

```tsx
<Pressable
  style={({ pressed }) => [styles.listItem, pressed && { opacity: 0.6 }]}
  onPress={item.onPress}
  android_ripple={{ color: colors.textSecondary }}
>
```

10. **Use a `textSecondary` color for icons and chevrons**. Define it at the top of the component:

```typescript
const textSecondary = isDark ? "#8E8E93" : "#6E6E73";
```

Destructure `isDark` from `useThemedStyles()` alongside the existing `styles`.

Keep all existing functionality: badge display, footer version text, logout alert, feedback alert, developer items gated by `diagnosticsEnabled`.

Do NOT change the list structure (keep `FlatList`) — only add icon and chevron rendering to each row, and fix the hidden-tab push path.
