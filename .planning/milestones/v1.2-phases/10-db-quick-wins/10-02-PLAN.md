Eliminate all N+1 upsert loops across src/db/helpers/ by converting every batch write function to a single-statement `INSERT ... ON CONFLICT DO UPDATE` call. Write test assertions first so regressions are caught immediately.

Output: Two new test files (batch assertions), nine updated helper files (all N+1 loops removed).
